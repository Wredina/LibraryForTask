- help() выводит подсказку об объекте, который передается в
качестве аргумента. более продвинутая версия dir().

help() без аргументов запускает интерактивный режим. В нём
можно указывать имена зарезервированных слов, встроенных функций,
модулей и получать справочную информацию

```py
help(str)
```

```py
help()
```

Для выхода из режима справки используйте команду quit

- isinstance(object, classinfo) - Принимает на вход объект и класс и возвращает истину, если объект является экземпляром прямого или косвенного подкласса


- hash(object) - Возвращает hash объекта в виде целого числа. Если объект не изменяемый, то можно вычислить hash, а если он изменяемый - hash не вычисляется.

```py
x = 42
y = 'text'
z = 3.1415
print(hash(x), hash(y), hash(z))
my_list = [x, y, z]
print(hash(my_list)) # получим ошибку, т.к. list изменяемый
```

> TypeError: unhashable type:
'list'

# Математические

- Для приведения строки к числу int() — целый тип и float()

```py
age = int(input('Ваш возраст: '))
how_old = age - 18
print(how_old, "лет назад ты стал совершеннолетним")
```
>
>Ваш возраст: 33
>
>15 лет назад ты стал совершеннолетним

- int(x, base=10) - Первый аргумент — объект, который мы хотим преобразовать в число, Второй аргумент указывает на основание
системы счисления. По умолчанию используется десятичная система, её можно не
указывать. Для base допустимы значения от 2 до 36, т.е. от двоичной до тридцати
шестиричной системы счисления

```py
x = int("42")
y = int(3.1415)
z = int("hello", base=30)
print(x, y, z, sep='\n')
```

>42
>
>3
>
>14167554

- _ упрощает чтение больших чисел.

```py
num = 7_901_123_456_789
```

>7901123456789

- float(x) - возвращает число с плавающей запятой, составленное из чиса или строки x. могут приводить к погрешностям
вычисления

```py
print(0.1 + 0.2)
```

- bin(x) - преобразует целое число в двоичную строку с префиксом "0b" возвращают строку

- oct(x) - преобразует целое число в восьмеричную строку с префиксом "0o" возвращают строку

- hex(x) - преобразует целое число в шеснадцатиричную строку с префиксом "0x" возвращают строку

```py
num = 2 ** 16 - 1
b = bin (num)
o = oct (num)
h = hex (num)
print(b, o, h)
```

>0b1111111111111
>
>0o177777
>
>0xffff

- complex([real[,imag]]) - комплексное число из действительной real и мнимой imag частей

Комплексным числом Z называется выражение вида: a + bi, где a и b - действительные числа, а символ i2 = -1. Число a - действительная часть, bi - мнимая часть, i - мнимая единица. C - множество комплексных чисел.

для обозначения
мнимой единицы использует букву "j"

```py
a = complex(2,3)
b = complex('2+3j')
print(a,b,a == b, sep='\n')
```

>(2+3j)
>
>(2+3j)
>
>True
>

- abs(x) — возвращает абсолютное значение числа x, число по модулю. (возвращает всегда только положительное число)

```py
x = -42
print(abs(x))
```

>42

- divmod(a, b) — функция принимает два числа в качестве аргументов и
возвращает пару чисел — частное (результат целочисленного деления a на b) и остаток от целочисленного деления.
Аналогично вычислению a // b и a % b.

```py
a = 45
b = 5
print(divmod(a, b))
```

>(8,2)

- pow(base, exp[, mod]) — при передаче 2-х аргументов возводит base в
степень exp. При передаче 3-х аргументов, результат возведения в степень
делится по модулю на значение mod.

```py
a = 45
b = 5
print(pow(a, b))
print(pow(a, b, 10))
```

>130691232

- round(number[, ndigits]) — округляет число number до ndigits цифр
после запятой. Если второй аргумент не передать, округляет до ближайшего
целого.

```py
print(round(3.141_592_653_589, 3))
```

> 3.142
